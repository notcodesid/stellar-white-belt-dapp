# Stellar White Belt dApp (React + Vite)

Level 1 white belt app that runs on **Stellar testnet** and implements:

- Freighter wallet connect + disconnect flow
- Testnet XLM balance fetch and display
- XLM payment submission on testnet
- Transaction feedback (success/failure + hash)

## Run locally

```bash
npm install
npm run dev
```

Open: <http://localhost:4173>

## Prerequisites

1. Install Freighter extension.
2. In Freighter, switch network to **Testnet**.
3. Fund your account via Friendbot.
